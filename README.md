# Villa Management System

A full-stack real estate management platform for property listings and bookings built with Django and Python.

## Features
- Property listings, bookings, and builder approvals
- Role-based dashboards for Admin, Builder, and User
- Property CRUD operations and site visit scheduling
- Admin control panel to manage registrations and monitor bookings

## Tech Stack
- Backend: Python, Django
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Database: MySQL
